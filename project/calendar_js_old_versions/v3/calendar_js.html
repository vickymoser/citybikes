Calendar JS/DOM v3.0

[Calendar JS by derletztekick.com]

Folgender Code muß an der Stelle in der gewünschten (X)HTML-Seite eingefügt werden, an der Calendar JS/DOM dargestellt werden soll:

<div id="calendar">[Calendar JS by derletztekick.com]</div>

Ferner sind im Kopfbereich des Dokumentes sowohl das Script als auch die CSS Datei einzubinden:

<link rel="stylesheet" media="screen" href="./calendar_js.css" type="text/css" />
<script type="text/javascript" src="./calendar_js.js"></script>
